Big Data Analytics

When we talk about big data we need to first understand what is actually known and identified as big data

As an evolving field we have ever chaniging characteristics but for now we can say that we have 6Vs but there are people who say there is 10+ Vs

Volume(Scale of data), Velocity(Analysis of streaming data), Variety(different forms of data), Veracity(uncertainty of data), Valence(connectedness of data), Value(benefits of the data)

Volume

Amount of data generated

  • Can it be stored?
  • Can it be used?
  • Can it be analysed?

Velocity

  • How fast data is generated
  • How fast data needs to be moved?
  • How fast the data needs to be analyzed?

Variety

  • Structred and Unstructured Data
  • Different Formats
  • Varying Sources
  • Vendor specific data
  • Different measuring units
  • Incomplete Data

Variety

  • Degree to which data is accurate
  • Precise and accurate
  • Unbiased
  • Bug Free
  • Secured
  • Falsification Free
  • Recent
  • Human Error Free

Valence

  • Data connectedness
  • Relationship between data sets
  • Should increase over time to have denser connections
    • Denser connections causes more complex data explorations, emergent behavior analysis and group event detections

Value

  • What is the problem that is being solved?
  • What new knowledge is obtain from the data?
  • Results versus Investement ROI
  • Automation based on data discoveries
  • Trigger Innovation
  • Trigger new products or services
